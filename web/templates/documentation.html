JLM API Documentation

This is a demo project which fetches the data from a website through an API call and inserts it into a postgres Database.

The Stock data for sp 500 companies are available here for the period between {January 2000} and {Yesterday}.

Calls can be made to present the data in any currency.

Examples how to query the data:

  • Plots
  • Data

Plots

The Plot endopoint enables the visualization of a trendline plot of any company in any given timeframe by typing the following URL on the browser, replacing the keywords encapsulated by < and > with the desired information:

/plots/<ticker>/<from_date>/<to_date>

Data

The stock / currency / data endpoint enables the retrieval of any stock in any currency for any given timeframe. To do so, one should type the following URL on the browser, replacing the keywords encapsulated by < and > with the desired information:

/endpoints/stock_data/<currency_symbol>/<ticker>/<from_date>/<to_date>

For example, to retrieve stocks from Microsoft in EUR currency during the year of 2019, one should paste the following:


/endpoints/stock_data/EUR/MSFT/20170000/20171231
